# tive-vue2-mobile-demo

vue2.0+vantUI移动端模板Demo

## 使用命令

```sh
# 启动本地服务
npm start

# 动态监听 vue.config.js 重启服务
npm run dev

# 打包
npm run build

# 打包后生成zip压缩包
npm run zip

# 生成打包报告
npm run build:az
```
